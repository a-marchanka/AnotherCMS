Datenschutz

Die Nutzung unserer Webseite erfogt ohne Angabe personenbezogener Daten. Die Eingabe der personenbezogenen Daten (beispielsweise Name, Anschrift oder E-Mail-Adressen) erfolgt auf freiwilliger Basis. Diese Daten werden an Dritte nicht weitergegeben.

Hiermit wird der Nutzung von im Rahmen der Impressumspflicht veröffentlichten Kontaktdaten durch Dritte zur übersendung von Werbung und Informationsmaterialien widersprochen.

Arten der verarbeiteten Daten

Für den Betrieb der Webseite werden folgende Daten verarbeitet und als Log Dateien gespeichert:

- IP Version (IPv4 / IPv6)
- Timestamp (Zeitstempel)
- Query Name, Query Type, Query Class
- Response Type (normal, timeout, blocked)
- Response Code, Response Size, Records in Response, Response Time in Milliseconds

Nach 7 Tagen erfolgt eine automatische Archivierung und nach 3 Monaten eine automatische Löschung der Archive Dateien auf dem Webserver.

Ihre Kontaktanfragen und Aufträge werden automatisch verarbeitet und in der Datenbank gespeichert.

Cookies und Inhalte der Dritte

Auf unserer Webseite werden keine Cookies verwendet.

Auf unserer Webseite sind keine Drittdienste angebunden, wie z.b. Datentracker, Werbedienste oder Soziale Medien.
